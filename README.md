## Overview
This project simulates a radar-like system using an ultrasonic sensor mounted on a servo motor to scan the surrounding environment. It measures object distance and angle for basic obstacle detection.

## Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Jumper Wires

## Working Principle
The ultrasonic sensor rotates using a servo motor and measures distance at different angles. This creates a radar-like scanning system.

## Features
- Real-time object detection
- Distance measurement
- Rotational scanning

## How to Run
1. Connect components properly
2. Upload code to Arduino
3. Open Serial Monitor
4. Observe scanning

## Applications
- Obstacle detection
- Robotics navigation
- Basic surveillance

## Learning Outcomes
- Embedded systems basics
- Sensor integration
- Hardware-software interaction
